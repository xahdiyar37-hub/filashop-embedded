## 二、实验方案设计

### 实验矩阵（正交实验法）

采用 **L9（3^4）正交表**，4 因素 3 水平，共 9 组实验 + 3 组验证实验 = **12 组**。

## 三、工艺参数（固定条件）

所有实验组使用统一的工艺参数（消除工艺变量干扰）：

| 参数 | 设定值 | 说明 |
|------|--------|------|
| 预干燥 | PLA: 60°C × 4h；PBAT: 50°C × 3h；PEG: 不干燥 | 除湿至含水率 <0.1% |
| 料桶温度 | **195°C** | 比 PLA 单独加工高 15°C，保证 PBAT 充分熔融 |
| 螺杆速度 | **26 RPM** | 降速保证混炼均匀 |
| 牵引速度 | **32 mm/s** | 与螺杆速度匹配，控制直径 1.75mm |
| 冷却水槽温度 | 常温（25°C） | |
| 抗氧剂 1010 | 固定添加 0.2% | 所有组统一 |

## 四、评价体系

### 1. 感官评价（目视+触感）

| 指标 | 评分 1-5 分 | 标准 |
|------|------------|------|
| **光泽度** | 1=哑光 5=镜面 | 60°光泽度计辅助 |
| **丝绸感** | 1=无 5=强烈丝绸质感 | 与市售 silk PLA 对比 |
| **表面光滑度** | 1=粗糙 5=极光滑 | 手指触摸 + 放大镜观察 |
| **颜色均匀性** | 1=斑驳 5=均匀 | 目视 |

### 2. 力学性能（可选）

| 指标 | 测试方法 | 目标值 |
|------|---------|--------|
| 拉伸强度 | ASTM D638 | ≥ 30 MPa |
| 断裂伸长率 | ASTM D638 | ≥ 5% |
| 层间结合力 | 打印 10×10×10mm 方块，手撕测试 | ≥ 普通 PLA 80% |

### 3. 打印测试

打印标准化测试模型（温度塔 + 光泽样片），评价：
- 190°C / 200°C / 210°C 三档温度打印
- 外墙光泽均匀性
- 层附着力

### 4. 直径一致性

| 指标 | 测量方法 | 合格范围 |
|------|---------|---------|
| 直径 | 游标卡尺，每隔 1m 测一次 | 1.70-1.80mm |
| 椭圆度 | 同截面测两个方向 | 差值 <0.05mm |

## 六、操作流程

### Day 1-2：材料准备
1. 采购所有原材料和耗材
2. PLA 颗粒 60°C 干燥 4 小时
3. PBAT 颗粒 50°C 干燥 3 小时
4. 按实验分组预称量每组分装到铝箔袋（每袋 200g 总量），贴标签密封

### Day 3-5：挤出实验（E1-E9）
每组流程：
1. 开机预热料桶至 195°C，稳定 15 分钟
2. 加入对应配方的预混料 200g
3. 螺杆 26 RPM 运转，观察出料状态
4. 牵引机 32 mm/s，冷却水槽常温
5. 绕线，每米标记直径测量点
6. 取 10m 样品用于打印测试
7. 填写实验记录卡
8. 清洗螺杆（空转 PLA 3 分钟），准备下一组

### Day 6-7：数据分析
1. 汇总 9 组评分到汇总表
2. 正交分析：计算各因素各水平的均值和极差
3. 确定最优组合 → 安排 V1 验证实验
4. 同步进行 V2、V3 对照实验

### Day 8-9：验证与打印测试
1. 挤出 V1/V2/V3 各 200g
2. 3D 打印机打温度塔（190/200/210°C）
3. 打印光泽样片（10×10×3mm 方块，外墙 3 圈）
4. 光泽度计测量、拍照存档
5. 与市售 silk PLA 样品对比

### Day 10：报告撰写
1. 正交分析表
2. 最优配方确定
3. 工艺参数推荐
4. 成本核算
5. 下一步优化方向

## 八、注意事项

1. **干燥是关键** — PLA 吸潮后挤出会产生气泡，表面起泡完全无法评价光泽。务必充分干燥
2. **珠光粉分散** — 珠光粉容易团聚，建议先用少量 PLA 颗粒和珠光粉在密封袋中摇匀（母粒法），再加入主料
3. **螺杆清洗** — 每组实验之间必须用纯 PLA 过料清洗 3 分钟，避免上一组残留干扰
4. **温度控制** — PBAT 比例 >10% 时可能需要微调温度（±5°C），记录实际温度
5. **安全** — PEG 粉末和珠光粉吸入有害，操作时佩戴口罩和手套
6. **存档** — 每组挤出样品取 1m 保留，用标签贴好实验编号，方便后续对比

## 九、预期结果

基于文献和行业经验，预期：

| 因素 | 预期影响 |
|------|---------|
| PBAT 10-15% | 丝绸光泽最明显；5% 效果弱，15%+ 可能过软 |
| PEG 2% | 流动性最佳，光泽提升明显；3%+ 可能太粘 |
| 珠光粉 1% | 增强丝绸感；2% 可能影响打印性能 |
| 相容剂 1-2% | PBAT ≥10% 时明显改善分散和光泽均匀性 |

**预测最优配方**：E4 或 E5 附近（PBAT 10%、PEG 2%、珠光粉 1-2%、相容剂 0-2%）
